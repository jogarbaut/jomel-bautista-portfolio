Projects

MERN Keeper

This is a web application for note keeping inspired by the features of Google Keep. Users have the ability to create, update, delete, and search notes. Based on the importance of the note, users can also pin/unpin notes which visually separates them.

My project includes a bonus feature of Opitcal Character Recognition powered by the Tesseract.js library. In the demo, an image of a pancake recipe can be seen being uploaded and the recipe text automatically being filled into the body of the note.

Technologies: React, React Custom/Advanced Hooks, MongoDB, Mongoose, Node.js, Express, JSON WebTokens, Tesseract.js, Git, and Github
