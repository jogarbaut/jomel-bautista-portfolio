Healthability Client Portal Web Application

I created a Full-Stack MERN Multi-Page Web Application that serves as a portal for a therapist and clients to keep track of what exercises or tasks are assigned. Often times we leave a session with a therapist with a piece of paper of exercises to do but no system to track if it gets done.

The application integrates Redux Toolkit and RTK Query for to support data fetching and caching performance. CRUD functionality is achieved with MongoDB and Mongoose.

Authentication and authorization is used to establish user-based protected routes. This is made possible with backend middleware which applies Bcrypt and JSON Webtokens.

React Router is also used with the useParams hook to handle routing and editing of specific data objects.

Technologies: MongoDB, Mongoose, Express.js, React, React Router, Redux, RTK Query, Node.js, Bcrypt, Cookie-Parser, JSON Webtokens, HTML, CSS, JvaScript, CSS, Git, and Github
