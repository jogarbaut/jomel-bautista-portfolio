Featured Projects

San Diego Relentless Youth Basketball Organization Web Application

I developed a Full-Stack MERN Application for a youth basketball organization to display organization news, rosters, and other relevant information. It includes a custom content management interface for organization admins to perform CRUD functionality of 5 data models using the MERN stack with supporting technologies such as Mongoose and Axios.

A challenge was to find a solution to handle media upload/storage due to the unknown quantity of uploads. To overcome this challenge, I integrated the Cloudinary API protocol to the backend endpoints to handle image upload/storage.

Another challenge protecting the application with authentication/authorization to prevent non-admin users from updating content. This was accomplished via middleware with Bcrypt, JWT, Cookie, and user context.

Technologies: MERN Stack, React, MongoDB, Mongoose, Node.js, Express.js, JSON WebTokens, Git, and Github

React Keep Web Application

I built a React Single Page Web Application by cloning the features of the Google Keep web application.

User interactions are managed through the application state to conditionally render components and their style (such as the color of a note). The user interface also allows users to separate notes by pinning and unpinning or switching to a grid or stacked view.

I achieved efficient search capabilities using React context and JavaScript array filtering methods.

My project includes a bonus feature of image to text conversion powered by the Tesseract.js library.

Technologies: React, HTML, CSS, JvaScript, CSS, Git, and Github
